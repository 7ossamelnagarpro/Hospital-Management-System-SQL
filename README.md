# 📌 Project Overview

The Hospital Management System is a relational database project designed to manage hospital operations including:

- Patients Management
- Doctors and Departments
- Appointments
- Medical Records
- Prescriptions
- Medicines Inventory
- Room Assignments
- Billing System

The project demonstrates database design, SQL development, and business analysis using Microsoft SQL Server.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Design a normalized relational database.
- Create relationships between hospital entities.
- Implement constraints to maintain data integrity.
- Build analytical queries for reporting.
- Create reusable database objects such as:
  - Views
  - Stored Procedures
  - Functions
  - Triggers
  - Indexes
